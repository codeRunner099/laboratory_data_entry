Deutsch

Offline-Tool zur Verwaltung von Laboraufträgen, Proben und Probenparametern.
Die Daten werden komplett im Browser über IndexedDB gespeichert.
Enthält Tabellen mit Inline-Bearbeitung, Detail-Popups und JSON-Export.
______________

English

Offline tool for managing laboratory orders, samples, and sample parameters.
All data is stored locally in the browser using IndexedDB.
Includes inline table editing, detail popups, and JSON export.
